# PatronusDemo
**Coding Challenge Task**

A basic app to show the list of the Patronus device holders and their details. 


#### Technologies and Tools 

- Kotlin
- MVVM
- Clean Architecture
- Multi Module
- Hilt 
- Retrofit
- Coroutine
- Jetpack Compose
- Jetpack Compose Navigation Component
- Google Map SDK
- Coil
- Mockito
- Kotest
