# PatronusDemo
**Coding Challenge Task**

A basic app to show the list of the Patronus device holders and their details, which consists of:

MVVM

Clean Architecture

Multi Module


#### Libraries and Tools 

- Hilt 
- Retrofit
- Coroutine
- Jetpack Compose
- Jetpack Compose Navigation Component
- Google Map SDK
- Coil
- Mockito
- Kotest
